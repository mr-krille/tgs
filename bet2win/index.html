About this project

A well-known sports betting platform, Bet2Win was previously only marginally active online. The Gentleman Studio took the company a big step forward by creating a completely new concept for online betting websites, and designed a simple interface to display live results as well as sales-oriented elements. There is also a native web application which allows consistent quality access across all devices.

The Challenge

The implementation of both technology and design on this project were fairly complex, involving very large data sets, and demanding extreme safety & sales oriented design solutions. Despite this, we managed to create a comfortable interface for the user exceeding all expectations.

The Facts

  • Designed in spring 2014
  • Number of finished screens: about 50
  • Responsive on tablet & notebooks
  • Individual app interface concept
